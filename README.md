# ecommerce

Monorepo local que unifica el frontend y el backend del proyecto.

## Estructura

- `frontend/`: aplicacion Angular
- `backend/`: API en TypeScript/Node.js

## Puesta en marcha

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm start
```

## Notas

- El remoto actual sigue apuntando a `ZapasPro_Front` hasta que renombres el repositorio en GitHub.
- Cuando el repo remoto se llame `ecommerce`, actualiza `origin` con la nueva URL.
